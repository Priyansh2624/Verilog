# Verilog Projects

A collection of Verilog HDL designs created while learning Digital Electronics, Digital Design, and VLSI Design concepts.

This repository contains implementations of fundamental combinational and sequential circuits using Verilog HDL. The goal of this repository is to build a strong foundation in RTL design and hardware description languages before moving on to more advanced projects such as UARTs, FIFOs, processors, and ASIC design flows.
---

## Learning Objectives

Through these projects I aim to:

- Develop proficiency in Verilog HDL.
- Understand combinational and sequential logic design.
- Learn RTL design methodologies.
- Build a strong foundation for FPGA and ASIC design.
- Prepare for advanced VLSI and Computer Architecture projects.
